ତୃତୀୟ ଅଧ୍ୟାୟ: ସମାନ୍ତର ପ୍ରଗତି (Arithmetic Progression)

୩.୧ ଅନୁକ୍ରମ (Sequence)

  • ଏକ ନିର୍ଦ୍ଦିଷ୍ଟ ନିୟମକୁ ଭିତ୍ତି କରି ନିର୍ଦ୍ଦିଷ୍ଟ କ୍ରମ (Order) ରେ ଥିବା ସଂଖ୍ୟାସମୂହକୁ ଏକ ଅନୁକ୍ରମ (Sequence) କୁହାଯାଏ [3]।
  • ଅନୁକ୍ରମରେ ଥିବା ପ୍ରତ୍ୟେକ ସଂଖ୍ୟାକୁ ଗୋଟିଏ ଗୋଟିଏ ପଦ (Term) କୁହାଯାଏ। ଏଗୁଡ଼ିକୁ t1,t2,t3,... ଭାବେ ସୂଚାଯାଏ [4]।
  • ଅନୁକ୍ରମର n-ତମ ପଦକୁ tn ଦ୍ୱାରା ସୂଚାଯାଏ ଏବଂ ଏହାକୁ ସାଧାରଣ ପଦ (General term) କୁହାଯାଏ [4]।

୩.୨ ସମାନ୍ତର ପ୍ରଗତି (Arithmetic Progression - A.P.)

  • ଯଦି ଏକ ଅନୁକ୍ରମର ପ୍ରତ୍ୟେକ ପଦରୁ (ପ୍ରଥମଟିକୁ ଛାଡ଼ି) ତାହାର ପୂର୍ବ ପଦର ବିୟୋଗଫଳ ସର୍ବଦା ସମାନ ହୁଏ, ତେବେ ତାହାକୁ ସମାନ୍ତର ପ୍ରଗତି (A.P.) କୁହାଯାଏ [5]।
  • ଏହି ସ୍ଥିର ବିୟୋଗଫଳକୁ ସାଧାରଣ ଅନ୍ତର (Common difference) କୁହାଯାଏ ଏବଂ ଏହାକୁ 'd' ଦ୍ୱାରା ସୂଚିତ କରାଯାଏ [5]।
  • d=t2−t1=t3−t2=... [5]
  • ଯଦି ପ୍ରଥମ ପଦ 'a' ଏବଂ ସାଧାରଣ ଅନ୍ତର 'd' ହୁଏ, ତେବେ A.P. ର ସାଧାରଣ ରୂପ ହେଉଛି: a,a+d,a+2d,... [6]।
A.P. ର n-ତମ ପଦ ସୂତ୍ର:

tn = a + ( n − 1 ) d

୩.୩ ପ୍ରଥମ n ସଂଖ୍ୟକ ପଦର ଯୋଗଫଳ (Sn)

ଏକ A.P. ର ପ୍ରଥମ n ସଂଖ୍ୟକ ପଦର ସମଷ୍ଟି (Sn) ନିର୍ଣ୍ଣୟ ପାଇଁ ଦୁଇଟି ମୁଖ୍ୟ ସୂତ୍ର ଅଛି [7]:

ସୂତ୍ର ୧:
Sn = n2 [ 2a + ( n − 1 ) d ]
ସୂତ୍ର ୨ (ଯଦି ଶେଷ ପଦ l ଜଣାଥାଏ):
Sn = n2 ( a + l )

କିଛି ଗୁରୁତ୍ୱପୂର୍ଣ୍ଣ ଯୋଗଫଳ ପରିଣାମ:

  • ପ୍ରଥମ n ସଂଖ୍ୟକ ଗଣନ ସଂଖ୍ୟାର ସମଷ୍ଟି: n(n+1)2 [8, 9]
  • ପ୍ରଥମ n ସଂଖ୍ୟକ ଅଯୁଗ୍ମ ଗଣନ ସଂଖ୍ୟାର ସମଷ୍ଟି: n2 [10]
  • ପ୍ରଥମ n ସଂଖ୍ୟକ ଯୁଗ୍ମ ଗଣନ ସଂଖ୍ୟାର ସମଷ୍ଟି: n(n+1) [11]
  • ପ୍ରଥମ n ସଂଖ୍ୟକ ଗଣନ ସଂଖ୍ୟାର ବର୍ଗର ସମଷ୍ଟି: n(n+1)(2n+1)6 [12]
  • ପ୍ରଥମ n ସଂଖ୍ୟକ ଗଣନ ସଂଖ୍ୟାର ଘନର ସମଷ୍ଟି: [n(n+1)2]2 [13]

୩.୪ ସମାନ୍ତର ମଧ୍ୟକ (Arithmetic Mean - A.M.)

  • ଦୁଇଟି ସଂଖ୍ୟା a ଓ b ର ସମାନ୍ତର ମଧ୍ୟକ ହେଉଛି a+b2 [14]।
  • ଯଦି a,x,b ଏକ A.P. ରେ ରହନ୍ତି, ତେବେ x କୁ a ଓ b ର ସମାନ୍ତର ମଧ୍ୟକ କୁହାଯାଏ [14]।
  • ଯଦି ଦୁଇଟି ସଂଖ୍ୟା ମଧ୍ୟରେ n ସଂଖ୍ୟକ ସମାନ୍ତର ମଧ୍ୟକ ରଖାଯାଏ, ତେବେ ସାଧାରଣ ଅନ୍ତର d=b−an+1 ହୁଏ [15, 16]।
